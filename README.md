# NDIS Support Coordinator Document Generator

An AI-powered system to help NDIS Support Coordinators generate professional reports quickly and compliantly. Supports Change of Circumstance, Progress, Implementation, Support Plans, and Assessments.

## ✨ Key Features
- **Coordinator & Client Management**: Profiles, credit tracking, client folders, NDIS details, goals, and notes.  
- **Document Handling**: Upload OT reports, plans, and medical files (PDF/DOCX/TXT). AI-powered parsing and summarization.  
- **Template Library**: Change of Circumstance, Implementation, Progress, Plans, and Assessments.  
- **AI Generation**: NDIS-compliant, formal, person-centered language with evidence-based recommendations.  
- **Editing & Export**: Real-time editor, AI-assisted refinements, version control, and PDF/DOCX export.  
- **Credit System**: Pay-per-use model with transparent credit tracking and packages.
